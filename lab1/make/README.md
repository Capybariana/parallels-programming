Подсчет суммы синусов для float- и double-значений.

# Сборка

## Double

```
    make USE_DOUBLE=ON
```

## Float

```
    make
```

## Дополнительное

Введите `make clean`, чтобы удалить исполняемый файл.

# Запуск

```
    ./lab1
```

# Вычисленные значения сумм

```
    double: 0.00000000004895819654503
    float: -0.02778621949255466461182
```
